Gotham Ruby Conference
======================

Here you'll find the content and assets for [http://goruco.com][goruco].  Everything is
staticly generated using [Jekyll][jekyll].

If you want to build the site locally you'll need to clone the repository and run the following:

    $ sudo gem install mojombo-jekyll
    $ jekyll

[goruco]: http://goruco.com
[jekyll]: http://github.com/mojombo/jekyll
